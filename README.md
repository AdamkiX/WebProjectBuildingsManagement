**System zarządzania budynkami - projekt webowy**

W celu postawienia projektu należy w peirwszej kolejności mieć zainstalowane programy:
Docker Desktop: https://docs.docker.com/desktop/
InteliJ IDEA: https://www.jetbrains.com/idea/

Po ich instalacji można pobrać repozytorium na komputer.
Projekt należy otworzyć w IDE i wejść w folder zawierający backend, tam znajduje się Read.me informujące o postawieniu kontenera. Należy się z nim zapoznać.
Następnie nalezy wejść w folder zawierający frontend i zapoznać się z Read.me informującym o postawieniu frontendu, również należy się z nim zapoznać.
Kolejno należy utworzyć połączenie do bazy MySQL zapoznając się z backendową konfiuracją Dockera - tam są opisane wszystkie mapy portów dla aplikacji i bazy.
W celu testu aplikacji można wykorzystać testową bazę danych, wystarczy uruchomić na niej querry, aby wygenerować dane.
